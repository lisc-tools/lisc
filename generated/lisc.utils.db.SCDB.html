    lisc.utils.db.SCDB¶

    class lisc.utils.db.SCDB(base=None, generate_paths=True, structure={1: {'base': ['terms', 'logs', 'data', 'figures']}, 2: {'data': ['counts', 'words']}, 3: {'words': ['raw', 'summary']}})[source]¶

    Database object for a SCANR project.

    Notes

    The default set of paths for SCDB is:

    Level 1: Base

    terms

    Terms files.

    logs

    Logs files.

    figs

    Figures files.

    data

    Data files.

    Level 2: Data

    counts

    Counts data files.

    words

    Words data files.

    Level 3: Words

    raw

    Raw words data files.

    summary

    Summary files for words data.
    Attributes
    pathsdict

    Dictionary of all folder paths in the project.
